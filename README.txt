Description
-----------
Ubimmo est un plugin WordPress qui simplifie l'importation automatique des annonces immobilières sur Ubiflow. Ce plugin est conçu pour les professionnels de l'immobilier qui cherchent à gagner du temps en automatisant le processus d'importation des annonces.

Features
--------
- Importation automatique des annonces immobilières sur Ubiflow.
- Personnalisation des paramètres d'importation pour répondre à vos besoins spécifiques.
- Facile à utiliser et à configurer pour une intégration fluide dans votre site WordPress.

Installation
------------
1. Téléchargez le fichier zip du plugin.
2. Accédez à votre tableau de bord WordPress et allez dans "Extensions" > "Ajouter".
3. Cliquez sur "Téléverser une extension" et sélectionnez le fichier zip que vous avez téléchargé.
4. Activez le plugin.
